B — QC process

Three layers of control from material to delivery.

01 / INPUT

Incoming material control

Every raw material batch (resin, pigment, solvent, additive) requires supplier COA + VPIC lab verification before warehousing. Non-conforming materials are returned.

02 / IN-PROCESS

In-process control

Hold-point inspections at key stages: post-mixing, post-milling, pre-filling. Samples compared against the customer's reference standard.

03 / OUTPUT

Finished goods control

Every batch tested against the spec sheet agreed with the customer. COA issued, retain samples kept 12 months for traceability.

C — COA documentation

Every batch comes with an official COA.

Certificate of Analysis (COA) is the document accompanying every VPIC delivery. Contains the following:

COA contents Details
Batch numberUnique traceable batch code with production date and shift identifier.
Production / expiry dateDate of manufacture and shelf life per formulation.
Technical specificationsAll spec parameters agreed with customer + actual measured values for the batch.
Test methodsReference standard (TCVN, ASTM, ISO) for each measurement.
QC personnelQC engineer who performed tests + QC manager approval.
Issue dateDate the COA was signed and stamped.
